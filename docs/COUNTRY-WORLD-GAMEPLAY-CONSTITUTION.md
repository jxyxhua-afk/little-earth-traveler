# 国家小世界玩法宪章

## 产品定位

《碰碰地球》不是国家百科，不是换皮沙盒，也不是把国家代表物摆在展台上的 3D 资料卡。

它应该是六个不同国家物理小游戏的集合。孩子从地球仪进入每个国家后，应该立刻感到：这里的物体不一样，玩法不一样，运动规律也不一样。

国家文化不靠文字背诵，而要通过操作体验被孩子记住。长城要能被修、防、撞；金字塔要能装、滑、堆；新干线要能赛、停；校车要能跑、投；回旋镖要能飞、返；冰屋要能撞、拆、重建。

## 国家设计硬标准

1. 每国必须有独立核心动词。
2. 每国必须有独立文化场景。
3. 每国必须有独立物理机制。
4. 每国必须有独立成功反馈。
5. 每国必须有独立学习目标。
6. 每国必须和已有国家不同。
7. 每国试玩前必须能用一句话说清楚：孩子在这里做什么，为什么这只属于这个国家。
8. 每国试玩后必须能回答：孩子是否主动重复同一个核心动作。

## 禁止事项

1. 禁止所有国家都是车、球、方块、斜坡。
2. 禁止只换颜色和背景。
3. 禁止用文字解释替代操作体验。
4. 禁止未试玩通过就扩国家。
5. 禁止把文化符号只做成不可玩的装饰。
6. 禁止所有国家共享同一套胜利反馈。
7. 禁止所有国家都靠“整理”作为唯一目标。
8. 禁止孩子不操作也能看完全部内容。

## 中国：修 + 防

文化符号：
长城、熊猫、竹林、烽火台、投石机。

玩法：
孩子不只是撞长城积木，而是在长城被撞乱后实时修复城墙，同时用熊猫投石机把软球投出去，阻挡来袭的小车或滚石。

物理机制：
碰撞冲击、材料强度、抛体运动。不同墙块有不同耐受力，竹子轻、城砖重，投石机发射角度影响落点。

成功反馈：
城墙补齐、烽火台亮起、熊猫挥手、获得长城守护章。

学习目标：
力会破坏结构，材料强度不同，抛出去的物体会沿弧线落下。

## 埃及：装 + 滑

文化符号：
金字塔、石块、尼罗河、莎草船、骆驼。

玩法：
孩子把石块装到小车或莎草船上，推到沙坡顶端，让石块滑下，再堆叠成金字塔。

物理机制：
斜坡摩擦、重心堆叠。石块越重越难推，坡越陡滑得越快，堆得太偏会倒。

成功反馈：
金字塔成型、太阳升起、骆驼铃响、获得金字塔建造章。

学习目标：
斜坡和摩擦会影响重物运输，重心不稳会倒塌。

## 日本：赛 + 停

文化符号：
新干线、富士山、樱花、信号灯、鸟居。

玩法：
孩子让新干线沿轨道竞速，在信号灯变化时控制刹车，精准停在站台。

物理机制：
惯性、刹车距离、轨道约束。速度越快，停车距离越长。

成功反馈：
停准站台、乘客小灯亮起、樱花飘落、获得新干线车长章。

学习目标：
速度越快，刹车距离越长；轨道会限制运动方向。

## 美国：跑 + 投

文化符号：
黄色校车、篮球、汉堡店、公路、自由女神像。

玩法：
孩子驾驶黄色校车绕过公路障碍，同时在移动中把篮球投向篮筐。

物理机制：
相对运动、抛物线、弹性碰撞。车在动时，球的落点会跟静止时不同。

成功反馈：
篮球进框、校车到站、汉堡店灯牌亮起、获得校车篮球章。

学习目标：
车在动时投出的东西落点会变化，弹性碰撞会改变方向。

## 澳大利亚：飞 + 返

文化符号：
回旋镖、袋鼠、草原、悉尼歌剧院、考拉。

玩法：
孩子投掷回旋镖击中草原目标，回旋镖飞回时让袋鼠跳起接住。

物理机制：
曲线飞行、侧向力、弹跳。投掷角度影响回旋路径，袋鼠跳跃要把握时机。

成功反馈：
回旋镖回到手里、袋鼠跳起、草原目标翻牌、获得回旋镖章。

学习目标：
力可以改变运动方向，跳跃需要时机。

## 北极：撞 + 拆

文化符号：
冰面、企鹅、冰块、雪坡、冰屋、北极光。

玩法：
孩子在低摩擦冰面上推冰块，撞击旧冰屋关键点，把它拆开，再用碎冰块对准形状重建冰屋。

物理机制：
动量传递、低摩擦、卡扣结构。冰块滑得远，撞到关键点会让结构松开，形状对上才稳。

成功反馈：
冰屋重建、北极光亮起、企鹅跳舞、获得冰屋工程章。

学习目标：
撞击关键点能破坏结构，形状对上才稳，低摩擦会让物体滑得更远。

## 开发顺序

1. 先做运动原型系统。
2. 再重做中国。
3. 再重做埃及。
4. 再做日本。
5. 再做北极。
6. 再考虑美国和澳洲。

## 每国开工前检查

1. 这个国家的核心动词是什么？
2. 它和现有国家的主要操作是否不同？
3. 它使用了哪个运动原型？
4. 文化符号是否已经变成玩法？
5. 孩子不看文字能不能理解要做什么？
6. 成功反馈是否属于这个国家？
7. 是否已经完成 3 到 5 分钟真实试玩？

未通过以上检查，不允许继续扩国家。
