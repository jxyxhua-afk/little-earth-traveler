# 美国 · 校车小镇探索

## 1. 关卡定位

国家：美国

关卡名称：美国 · 校车小镇探索

儿童认知目标：让孩子通过玩，认识美国的交通、建筑、运动、食物和电影文化符号。

核心体验：孩子驾驶一辆黄色校车，在一个 3D 美国小镇中穿行，发现 4-5 个美国标志物，并把这些发现点亮到“美国发现卡”里。

这不是“校车 + 篮球小游戏”。校车不是唯一主题，篮球也不是唯一目标。校车是孩子进入美国小镇的主物体，功能是带孩子去发现美国符号。

本关的正确方向是：

```text
开黄色校车
↓
逛美国小镇
↓
发现美国符号
↓
完成对应小互动
↓
点亮美国发现卡
```

孩子玩完后，应该能说出：

- 美国有黄色校车。
- 美国有自由女神像。
- 美国有好莱坞电影牌。
- 美国有人打篮球。
- 美国有汉堡 / 热狗摊和公路文化。

## 2. 美国标志物清单

### 黄色校车

孩子看到什么：一辆体积明确、颜色鲜明的黄色校车，有车窗、车灯、车轮、车门和车顶。

孩子能做什么：转方向盘、踩油门、让校车启动、转弯、停靠、轻轻撞开路上的小障碍。

对应认知：美国小朋友常坐黄色校车上学，黄色校车是美国儿童生活中非常容易识别的交通符号。

### 自由女神像

孩子看到什么：一个小广场里的自由女神像玩具，有火炬、皇冠和底座，造型要从剪影上能识别。

孩子能做什么：把掉落的火炬积木推回底座，或开校车停靠到广场，让火炬点亮。

对应认知：自由女神像是美国的著名地标。

### 好莱坞牌

孩子看到什么：山坡上的 HOLLYWOOD 字牌，前面被几个纸箱挡住。

孩子能做什么：开校车撞开纸箱，露出好莱坞牌；纸箱被撞后滑开、旋转、停下。

对应认知：美国有好莱坞，和电影文化有关。

### 篮球场

孩子看到什么：小型篮球场，有明显的篮板、篮圈、支架、篮球和地面球场颜色。

孩子能做什么：到达篮球场后，拖拽篮球，按抛物线投向篮筐；篮球落地会弹跳。

对应认知：篮球是美国常见运动文化符号之一。

### 汉堡 / 热狗摊

孩子看到什么：一个小吃摊，有汉堡、热狗、番茄酱瓶、纸盒或餐车。

孩子能做什么：推动或叠放汉堡食材积木，例如面包、生菜、肉饼；也可以把热狗推到小摊盘子上。

对应认知：汉堡、热狗是孩子容易识别的美国常见快餐符号。

### 66 号公路

孩子看到什么：一条穿过小镇的玩具公路，有 66 号路牌、斑马线、停车点和轻障碍。

孩子能做什么：沿路驾驶校车，停靠到不同发现点；撞开锥桶、纸箱、施工桶等轻物体。

对应认知：美国有公路文化，校车沿着公路带孩子去探索小镇。

### 摩天楼 / 金门大桥远景（可选）

孩子看到什么：远处的城市天际线、桥梁轮廓或红色大桥剪影。

孩子能做什么：作为远景认知符号，不一定需要复杂互动；可以在全部发现点完成后亮灯。

对应认知：美国也有大城市、高楼和著名大桥。

## 3. 小镇 3D 场景布局

本关不是一条单调马路，而是一个小型 3D 美国小镇玩具地图。

场景组成：

- 校车站：孩子起点，校车停在这里。
- 66 号公路：连接所有发现点的主路径。
- 自由女神像小广场：位于地图左上或中上区域。
- 好莱坞山坡：位于右上，字牌可被纸箱遮挡。
- 汉堡摊：位于下方或路边，适合叠食材小互动。
- 篮球场：位于右下或右侧终点区域。
- 远景城市天际线：放在地图背景边缘，不抢主玩法。

简单布局：

```text
                         好莱坞山坡
                    [纸箱] [HOLLYWOOD]
                            ▲
                            │
       自由女神像小广场 ── 66号公路 ── 篮球场
          [火炬积木]        [锥桶]       [篮筐]
                │            │
                │        汉堡 / 热狗摊
                │        [食材积木]
                │
              校车站
             [黄色校车]

远景：摩天楼 / 金门大桥轮廓 / 星条旗小旗帜
```

空间要求：

- 地图要有前后空间、体积、阴影和明确地面。
- 每个发现点要有足够间距，不能堆成展柜。
- 公路是引导线，不是唯一玩法主体。
- 篮球场不能和所有障碍混在一起，必须成为独立终点区域。

## 4. 核心流程状态机

```text
进入美国
↓
校车启动
↓
探索 66 号公路
↓
发现自由女神像
↓
撞开箱子露出好莱坞牌
↓
完成汉堡摊小互动
↓
到篮球场投篮
↓
点亮美国发现卡
↓
返回地球
```

状态说明：

1. 进入美国：镜头落到美国小镇俯视斜角，孩子第一眼看到校车站、路牌、自由女神像、篮球场等符号。
2. 校车启动：孩子用方向盘和油门让校车动起来，听到启动音。
3. 探索 66 号公路：校车沿主路行驶，路牌和轻障碍引导方向。
4. 发现自由女神像：校车停靠小广场，点亮或推回火炬积木。
5. 撞开箱子露出好莱坞牌：孩子看到纸箱被撞开，后面的 HOLLYWOOD 字牌亮起。
6. 完成汉堡摊小互动：孩子推或叠食材，完成一个简单快餐符号互动。
7. 到篮球场投篮：孩子拖拽篮球投篮，篮球按重力抛物线飞出并落地反弹。
8. 点亮美国发现卡：收集校车、自由女神像、好莱坞、汉堡摊、篮球场。
9. 返回地球：保留美国发现记忆，回到地球仪继续探索其他国家。

## 5. 可互动动作设计

### 驾驶

主物体：黄色校车。

孩子动作：转方向盘、踩油门，让校车在 66 号公路和小镇空间中移动。

反馈：车身轻微晃动，轮子转动，发动机声启动，松开油门后仍滑行一点再停下。

### 停靠

主物体：黄色校车。

孩子动作：把校车开到自由女神像广场、好莱坞山坡、汉堡摊、篮球场附近的发现点。

反馈：发现点亮起，出现图标反馈和短促提示音。

### 推撞

主物体：校车、纸箱、锥桶、施工桶、火炬积木。

孩子动作：用校车轻轻推开路上的小障碍，或把物体推到目标位置。

反馈：轻物体被撞后滑动、旋转、逐渐停下；校车速度下降或车身轻震。

### 拖拽

主物体：篮球、汉堡食材、火炬积木。

孩子动作：拖拽篮球蓄力，拖拽或推动食材到汉堡摊，拖拽火炬积木回自由女神像。

反馈：物体跟随手势移动，松手后受重力、摩擦或碰撞影响，不是瞬移。

### 投掷

主物体：篮球。

孩子动作：从篮球场拖拽篮球，松手后发射。

反馈：篮球有初速度、重力抛物线、旋转、落地反弹和命中反馈。

### 收集发现卡

主物体：美国发现卡。

孩子动作：完成某个发现点互动后，对应图标亮起。

反馈：图标从灰色变亮，播放发现音，场景中的对应标志物也亮一下。

## 6. 物理机制设计

### 校车惯性

校车必须有速度和摩擦。按下油门后加速，松开后不能立刻停住，要继续滑行一小段，再因摩擦减速。

设计要点：

- `position`
- `velocity`
- `acceleration`
- `friction`
- `maxSpeed`
- `mass`

### 车辆转向

方向盘控制校车朝向或速度方向。转向不能直接瞬移，必须有延迟和弧线感。

设计要点：

- `heading`
- `steering`
- `turnSpeed`
- 低速时转向弱，高速时转向更明显。

### 轻障碍推动

锥桶、纸箱、施工桶等轻物体要能被校车撞动。

设计要点：

- 每个轻障碍有 `position`、`velocity`、`mass`、`friction`、`rotation`、`angularVelocity`。
- 校车撞到障碍时，根据碰撞方向给障碍物 impulse。
- 障碍物滑动、旋转、逐渐停下。

### 纸箱碰撞

纸箱用于好莱坞牌发现点。它不是静态遮挡物，必须可以被撞开。

设计要点：

- 纸箱质量比锥桶略重。
- 撞击后可滑动并旋转。
- 纸箱离开遮挡区域后，HOLLYWOOD 字牌亮起。

### 篮球重力抛物线

篮球投掷必须符合基础力学，而不是按钮计数。

设计要点：

- 拖拽距离和方向决定初速度。
- 每帧应用重力：`velocity.y += gravity * dt`。
- 每帧更新位置：`position += velocity * dt`。
- 篮球沿抛物线飞向篮筐。
- 可显示简洁轨迹线或轨迹点，帮助孩子理解“球飞出去了”。

### 篮球落地反弹

篮球落地后要弹起并逐渐停下。

设计要点：

- 落地时 `velocity.y *= -bounce`。
- 水平速度受摩擦衰减。
- 篮球旋转与速度相关。
- 每次弹地播放橡胶球音效。

## 7. 声音反馈设计

声音必须和物体属性匹配，不能所有碰撞都用同一个音。

### 校车启动音

触发：第一次踩油门或按前进。

声音质感：低频、短促、带一点发动机颗粒感。

认知作用：孩子知道校车启动了。

### 轻喇叭

触发：校车到达发现点、发现卡点亮、或孩子主动点击喇叭图标。

声音质感：短促、明亮、不刺耳。

认知作用：强化校车身份。

### 轮胎摩擦音

触发：转向、急停、撞到边界后减速。

声音质感：轻微擦地声，不要尖锐。

认知作用：表现车和地面之间有摩擦。

### 碰撞音

触发：校车撞锥桶、纸箱、路障、施工桶。

声音质感：

- 锥桶：塑料轻响。
- 纸箱：纸板闷响。
- 路障牌：金属或木牌敲击声。
- 施工桶：空桶咚声。

认知作用：孩子通过声音区分材质。

### 发现标志物音

触发：自由女神像、好莱坞、汉堡摊、篮球场等图标点亮。

声音质感：清脆、短促、带一点奖励感。

认知作用：孩子知道“我发现了一个美国东西”。

### 篮球拍地音

触发：篮球落地、弹起、轻拍。

声音质感：橡胶球弹地声，不能像金属或纸箱。

认知作用：孩子知道篮球是会弹的球。

### 投篮命中音

触发：篮球穿过篮圈。

声音质感：清脆进球音 + 轻微欢呼或星星音。

认知作用：完成篮球场互动。

### 完成关卡音

触发：美国发现卡全部点亮。

声音质感：温和庆祝，不刺耳、不夸张。

认知作用：孩子知道美国小镇探索完成了。

声音启动规则：

- 浏览器不允许自动播放声音。
- 必须在首次点击、按键、触摸后解锁 AudioContext。
- UI 需要有声音开关。
- 静音后所有音效停止。

## 8. UI 设计

本关 UI 必须使用“美国发现卡”，不要只用任务计数。

发现卡内容：

```text
美国发现卡
□ 校车
□ 自由女神像
□ 好莱坞
□ 汉堡摊
□ 篮球场
```

实际画面应以图标为主：

| 发现项 | 图标方向 |
| --- | --- |
| 校车 | 黄色校车图标 |
| 自由女神像 | 火炬 / 女神像剪影 |
| 好莱坞 | HOLLYWOOD 山牌 |
| 汉堡摊 | 汉堡 / 热狗 / 小摊 |
| 篮球场 | 篮球 + 篮筐 |

UI 要求：

- 少文字，大图标。
- 不用“篮球 0/3”“障碍 0/3”作为主要反馈。
- 当前目标用场景高亮表达，例如发现点发光、路牌闪一下。
- 返回地球按钮固定左上。
- 声音按钮固定右上或角落。
- 方向盘和油门必须像真实物体，不能只写“前进”“左转”。
- 发现卡不能遮挡主要 3D 场景。
- 4 岁孩子不读字，也能看出要开校车、去发现图标。

## 9. 验收标准

### 国家认知

- 孩子不读文字，能不能看出这是美国？
- 孩子第一眼是否能看到校车、自由女神像、好莱坞、篮球场、汉堡摊等美国符号？
- 孩子玩完后能不能说出至少 3 个美国元素？

### 3D 与物体辨识度

- 校车是否是真 3D，有车身、车窗、车灯、轮子？
- 自由女神像是否能从剪影上识别？
- 好莱坞牌是否清楚，不只是装饰文字？
- 篮球场是否有篮板、篮圈、支架和篮球？
- 汉堡 / 热狗摊是否像食物摊，而不是普通盒子？

### 可互动性

- 至少 3 个物体是否可互动？
- 校车是否能推开轻障碍？
- 纸箱是否能被撞开并露出好莱坞牌？
- 汉堡食材或火炬积木是否能被拖动 / 推动？
- 篮球是否可拖拽投出？

### 物理反馈

- 校车是否有惯性和转向延迟？
- 轻障碍是否有速度、旋转和摩擦停下？
- 篮球是否有重力抛物线？
- 篮球是否会落地反弹？
- 碰撞是否不是简单 CSS 动画？

### 声音反馈

- 声音是否在首次点击 / 按键后可播放？
- 校车启动音是否存在？
- 不同物体碰撞音是否有区别？
- 篮球拍地音是否像橡胶球？
- 投篮命中音是否清楚？
- 静音按钮是否有效？

### 产品气质

- 是否不是普通网页小游戏？
- 是否不是“校车 + 篮球”的单薄任务？
- 是否像一个可探索的 3D 美国小镇？
- 是否围绕国家认知，而不是单纯闯关计分？

只有当孩子能通过这一关认识美国符号，并且愿意继续探索这些符号时，本关才算通过。
