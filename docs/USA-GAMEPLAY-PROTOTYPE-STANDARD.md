# 美国关卡样板标准

## 样板定位

美国 · 校车篮球街区不是一个孤立关卡，而是后续国家小世界的第一套完整样板。

这里的“样板”不是复制校车、篮球、公路，而是复制一套儿童 3D 物理关卡方法：

1. 孩子不看文字也知道自己要操作什么。
2. 核心物体有清楚外形，不靠标签解释。
3. 每个物体有自己的运动和声音反馈。
4. 玩法由国家文化符号转化而来。
5. 关卡有开始、探索、失败、重试、成功的完整闭环。

后续中国、日本、澳大利亚、北极等国家，必须复制这套结构标准，但不能复制美国玩法本身。

## 面向 4 岁儿童的 UI 原则

1. 文字只能辅助成年人理解，不能作为儿童主要操作入口。
2. 方向控制必须用孩子认识的物体隐喻，例如方向盘、踏板、拉杆、按钮、轨道、篮筐。
3. 操作按钮优先使用图形、形状、颜色、动效，不使用“前进”“左转”“开始任务”等文字作为唯一提示。
4. UI 不能遮住主要玩具场。
5. 关卡目标要通过场景布局表达：起点、路径、障碍、目标区必须一眼能看出来。

## 美国关卡当前核心规则

核心动词：
跑 + 投。

核心文化符号：
黄色校车、美国街区道路、篮球、篮筐、学校终点、自由女神像/城市装饰。

核心物理机制：
校车惯性、弹性碰撞、篮球抛物线、篮球弹跳。

儿童目标：
开校车，推开路上的小玩具，到球场后把篮球投进篮筐。

成功条件：
校车到达球场区域，并通过拖拽投篮让篮球穿过 3D 篮圈。

失败条件：
校车撞到重障碍或边界累计达到上限，或投篮机会用完。

重试方式：
使用重新开始按钮恢复状态；失败文案保持温和，不惩罚孩子。

## 关卡状态机

1. drive
   孩子驾驶校车。方向盘控制左右，油门踏板控制前进。校车有速度、摩擦、转向延迟和撞击反馈。

2. obstacle-play
   校车推动锥桶、纸箱、施工桶、路上的篮球等轻物体。每种物体必须有不同声音和运动反馈。

3. shoot
   校车到达球场后进入投篮。篮球成为主操作对象。

4. dribble
   孩子轻点或向下拖篮球，篮球弹起并落下，表现“拍球”感觉。

5. flying
   孩子朝篮筐方向拖拽松手，篮球按重力抛物线飞出。

6. success
   篮球进框，播放进球和完成反馈。

7. retry
   撞击或投篮未完成时，提示再试一次。

## 物体识别标准

每个主要物体至少满足三项：

1. 有真实轮廓：孩子能从外形认出它是什么。
2. 有功能部件：校车要有车窗、车灯、轮子；篮筐要有篮板、篮圈、支架；篮球要有球形和纹路。
3. 有对应运动：车会跑，球会弹，桶会滚，箱子会被推。
4. 有对应声音：不同材质和动作不能全部用同一种音效。
5. 有互动结果：碰撞后位置、旋转、速度或状态发生变化。

禁止：
1. 把文化符号做成静态摆件。
2. 用抽象盒子代替可识别物体。
3. 用文字说明替代外形和动效。
4. 所有碰撞都播放同一种声音。

## 美国关卡应保留的最小 UI

1. 返回地球。
2. 图形化方向盘。
3. 图形化油门踏板。
4. 声音开关。
5. 轻量进度提示。
6. 重新开始。

不应该增加：
1. 大段任务说明。
2. 复杂计分板。
3. 需要识字才能理解的按钮。
4. 调试框、红框、坐标、参数面板。

## 声音样板

美国关卡的声音系统是后续国家参考样板。

至少需要：

1. 主物体启动音：校车发动/起步。
2. 轻碰音：锥桶、篮球、小物体。
3. 材质音：纸箱、施工桶、路障牌要不同。
4. 操作音：方向盘、油门、投篮出手。
5. 物理音：篮球弹地、篮球进网。
6. 成功音：进球和完成。
7. 失败音：温和、短促，不恐怖。
8. 静音开关：必须有效。

浏览器声音必须由首次用户手势解锁，不能页面加载自动播放。

## 美国关卡验收清单

1. 进入后第一眼能看到 3D 玩具场，而不是网页平面。
2. 孩子能认出黄色校车。
3. 孩子能认出方向盘和油门。
4. 孩子能不用读字就让车动起来。
5. 校车有惯性，不是瞬移。
6. 锥桶、纸箱、施工桶、篮球至少三类物体可被推开。
7. 每类被推物体有不同声音或运动特征。
8. 篮筐是立体的，有篮板、篮圈、支架和网。
9. 篮球能拍起来。
10. 篮球能拖拽投出并按抛物线飞行。
11. 篮球能投进篮筐。
12. 投中后有明确成功反馈。
13. 失败后可以重新开始。
14. 移动端按钮不遮挡主要玩具。
15. 不出现调试框、红框和开发参数。

## 后续国家如何继承样板

继承的是流程，不是玩法：

1. 从国家玩法矩阵读取核心动词。
2. 把文化符号转成可操作玩具。
3. 为核心物体选择运动原型。
4. 设计图形化控制，不依赖文字。
5. 加入物体专属声音。
6. 建立失败、重试、成功闭环。
7. 做 3 到 5 分钟真实试玩。

每个国家必须回答：

1. 这个国家的核心动作是什么？
2. 孩子第一眼能不能认出主物体？
3. 孩子不读字能不能开始玩？
4. 物体是否有真实物理反馈？
5. 声音是否和物体材质/动作匹配？
6. 成功反馈是否属于这个国家？
7. 它和美国校车篮球是否明显不同？
